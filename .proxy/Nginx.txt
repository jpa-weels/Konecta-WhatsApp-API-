✅ OPÇÃO 1 — NGINX como Proxy Reverso (recomendado)
Ideal se você controla o servidor onde está o domínio.
🔹 1. Pré-requisitos
Domínio menflix.com.br apontando para o IP do servidor proxy

Ubuntu/Debian (exemplo)

sudo apt update
sudo apt install nginx -y
---------------------------------------------------------------------
sudo nano /etc/nginx/sites-available/menflix.com.br

server {
    listen 80;
    server_name menflix.com.br;

    client_max_body_size 20M;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_http_version 1.1;

        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        proxy_read_timeout 86400;
        proxy_send_timeout 86400;
    }
}
